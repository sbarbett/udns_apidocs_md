News and Updates

The News and Updates section provides a list of recent enhancements made to the REST API User Guide documentation that we want to make customers aware of. These updates may range from small clarification updates to new features becoming available via the REST API.

All of the updates found below will coincide with the Release Notes section, which can be found on the Home page of the Support portal.
