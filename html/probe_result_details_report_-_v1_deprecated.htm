Probe Result Details Report - v1 Deprecated

Unlike the various reports already documented, the Probe Result Details Report does not require the usage of the Report RequestID to generate the request for the report, and subsequently the return of the report. When a Probe Result Details Report is requested, the report is returned to the user immediately.

The version of the Probe Result Summary Report utilizing /v1 in the Method has been officially deprecated as of 2022-12-01.

We highly recommend that users begin to utilize the Probe Result Details v2 Report call as soon as possible to prevent any errors or confusion.

Requesting Probe Result Details Report

Method and URI:

GET https://api.ultradns.com/v1/reports/traffic_services/probe_result/details

 

Errors: If /v1 is present, Error 301 will be returned, indicating this API call is no longer supported.
